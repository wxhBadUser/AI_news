## 🔧 技术决策

### 1. 七维评分模型
| 维度 | 权重 |
|----|-----|
| 时间新鲜度 | 20% |
| 炸裂程度 | 25% |
| 重复度 | 15% |
| 争议性 | 15% |
| 可视化潜力 | 10% |
| 受众相关性 | 10% |
| 口播展开性 | 5% |

### 2. 去重机制
- 使用 MD5 哈希生成新闻唯一标识（取前12位）
- 基于 title + url 生成 hash

### 3. 三层 Fallback 采集策略
- 第一层：RSS 订阅（优先）
- 第二层：Browser 渲染（待实施）
- 第三层：Web Search 补偿（已实施）

### 4. 关键约束
- `exec` 工具要求直接运行脚本文件（不支持内联脚本）
- 禁止使用 `write` 工具写入包含中文的 JSON 文件
- 必须通过 Python 脚本执行所有数据持久化操作
